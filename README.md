# 🌸 Cycle Sync

Cycle Sync is a Flutter-based smart health tracking application designed to help women manage and monitor their menstrual health effectively.

---

## 🚀 Features

- 📅 Cycle Tracking – Track menstrual cycles and period history  
- 🔮 Ovulation Prediction – Predict ovulation and fertile windows  
- 📝 Symptom Logging – Record physical and emotional symptoms  
- 🔔 Smart Reminders – Get timely notifications for periods, medication, and appointments  
- 🚨 Emergency Guidance – Immediate actionable help for severe pain or irregularities  
- 🤖 AI Chatbot Assistant – Ask health-related questions and receive instant guidance  
- 👩‍⚕️ Doctor Access – Secure role-based data sharing with healthcare providers  

---

## 🛠 Tech Stack

- **Flutter**
- **Dart**
- **Firebase Authentication**
- **Cloud Firestore**
- **Firebase Storage**
- **Push Notifications**
